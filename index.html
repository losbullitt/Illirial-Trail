Cantebury Trails skeleton demo

Cantebury to New Isil - web prototype v0.4.2

Deploy this folder to any static host (Netlify, Vercel, GitHub Pages, Cloudflare Pages, S3, etc.); all asset paths are relative. For local development, run serve.ps1 in this folder (or py -3 -m http.server 8080 after cd here) and open http://127.0.0.1:8080/index.html. For a PC window (no browser), double-click desktop.bat or run py -3 desktop.py (one-time: py -3 -m pip install pywebview).
